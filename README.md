# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Flask, SQLite, and NLP.

## Features
- Resume Upload (PDF/DOCX)
- Skill Extraction
- ATS Score Calculation
- Resume Improvement Suggestions
- Job Description Matching

## Technologies Used
- Python
- Flask
- SQLite
- spaCy
- NLTK
- HTML
- CSS
- Bootstrap

## Installation

```bash
pip install -r requirements.txt
python app.py
